# Proyectos Universidad

Este repositorio contiene una recopilación de proyectos desarrollados durante mi formación universitaria en Informática Empresarial.

## Descripción

El objetivo de este repositorio es reunir diferentes proyectos académicos realizados en áreas como programación, desarrollo web, bases de datos, inteligencia artificial y lógica de programación.

Los proyectos incluidos reflejan mi avance en el uso de distintas tecnologías y mi capacidad para resolver problemas mediante soluciones de software.

## Contenido del repositorio

### Juegos

Proyectos enfocados en lógica de programación, estructuras de control, interfaces y desarrollo de videojuegos sencillos.

- BubbleShooter
- GatoAlgoritmos

### Proyectos IA

Proyectos relacionados con inteligencia artificial, procesamiento de información y modelos aplicados en contextos académicos.

- Identificador de dirección de zapatos
- LLM / Sistemas Expertos

### Proyectos varios

Proyectos académicos orientados al desarrollo de sistemas, interfaces y soluciones prácticas.

- Sistema de reservaciones

## Tecnologías utilizadas

- Java

- HTML
- CSS
- JavaScript
- SQL
- Inteligencia Artificial
- Programación orientada a objetos

## Objetivo profesional

Este repositorio forma parte de mi portafolio académico y profesional. Su propósito es mostrar proyectos realizados durante mi carrera y evidenciar mis conocimientos en desarrollo de software, bases de datos, lógica de programación y solución de problemas.
